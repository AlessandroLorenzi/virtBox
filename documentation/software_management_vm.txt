In cosa consiste:
Ovirt é un sistema per il management di macchine virtuali. 
É composto dall'immagine di un host che fornisce libvirt ed ospita le macchine virtuali.
Grazie a libvirt é possibile utilizzare Xen(linux/solaris), qemu/KVM, LXC, OpenVZ, User Mode Linux, VirtualBox
Console di gestione via web.

Gli utenti non hanno accesso all'hardware; automatizza la gestione del carico e semplifica la gestione delle macchine.
